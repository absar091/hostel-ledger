# Typography Update Summary

## iPhone-Style Typography Standards Applied

### Font Weights
- **Headers/Titles**: `font-black` (900 weight)
- **Section Labels**: `font-black uppercase tracking-widest`
- **Body Text**: `font-bold` (700 weight)
- **Amounts/Numbers**: `font-black tracking-tight tabular-nums`
- **Subtle Text**: `font-semibold` or `font-medium`
- **Card Titles**: `font-black tracking-tight`

### Text Sizes
- **Page Titles**: `text-3xl font-black tracking-tight`
- **Card Headers**: `text-lg` or `text-xl font-black tracking-tight`
- **Section Labels**: `text-xs font-black uppercase tracking-widest`
- **Amounts**: `text-2xl` to `text-4xl font-black tracking-tight tabular-nums`
- **Body**: `text-sm font-bold`

### Color Scheme
- Primary: `#4a6850` (greenish-gray)
- Text: `text-gray-900` for main, `text-[#4a6850]/80` for secondary
- Amounts: Use color coding (red for expenses, green for income)
