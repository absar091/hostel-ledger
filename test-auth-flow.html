5. Manual Testing Guide

Complete Flow Test:

  1. Open the main app in a new tab
  2. Navigate to the signup page
  3. Fill in basic information (name, email, university)
  4. Click "Continue" to go to password step
  5. Enter a strong password and confirm it
  6. Accept terms and privacy policy
  7. Click "Create Account"
  8. Check your email for verification code
  9. Enter the 6-digit code on verification page
  10. Should redirect to dashboard upon success

Error Scenarios to Test:

  • Try creating account with existing email
  • Try entering wrong verification code
  • Try resending verification code
  • Test password strength indicators
  • Test form validation
